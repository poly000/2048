以C语言重写的2048

从[2048](https://github.com/gabrielecirulli/2048)用C重写

操作方法：

W A S D 或 ↑ ← ↓ →

Q quit R reset

O save L load

编译（仅Windows）：

```
gcc 2048.c func.c -O3 -pipe -o 2048
```
