以C语言重写的2048

从[2048](https://github.com/gabrielecirulli/2048)用C重写

操作方法：

W A S D 或 ↑ ← ↓ →

Q 退出 R 重置

O 存档 L 读档

编译（仅Windows）：

```
gcc -std=c99 2048.c -o 2048
```
